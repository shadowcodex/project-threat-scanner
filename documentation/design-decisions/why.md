# Why does this exist? What purpose does it solve?

I started using this as my personal tool to check repos/github projects before using them myself. I am distrustful of any new tool no matter how popular it is.

I wanted a set of static scanners, and at first that's all it was. And I would also check snyk etc for this.

Then I added the agents once I learned more about various outcomes of agentic redteaming and hacking. So I added them to begin exploring agents' use in security landscapes when they have a whole host of tools and stuff available to them in the docker environment and a simple open ended question asked of them.

Hopefully folks can use this to:

A. Check stuff before they use it.
B. Check their own stuff and fix security issues with it.
C. Do cool things.
